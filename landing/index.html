How it works

Three operations. Any language.

Works from curl, Python, JavaScript, or any MCP client. No SDK required.

01
💾

Remember

POST any text to /v1/memory with a namespace. Memex embeds it semantically and stores it.

POST /v1/memory
{ "namespace": "...", "content": "..." }
02
🔍

Recall

Search with a natural language query. Get back the most relevant memories ranked by semantic similarity.

POST /v1/memory/search
{ "namespace": "...", "query": "..." }
03
📈

Persist

Memories accumulate over time. Importance scores keep the most useful ones surfaced. Low-value memories are pruned automatically.

GET /v1/memory/list
{ "namespace": "...", "limit": 20 }
